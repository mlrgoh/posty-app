Simple Node.js app that serves Markdown posts from the `posts/` folder. You can list posts, view them, and add new ones via a form.

## Setup

Run these commands in the `posty-app` folder:

```bash
npm install
npm start
```

Open `http://localhost:3000` in your browser. Use `New Post` to create a Markdown post.

Posts are saved as `.md` files in the `posts/` directory.
